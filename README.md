# Projeto - Operações no Mercado de Ações

Projeto desenvolvido como atividade prática de análise de dados utilizando planilhas.

Objetivo

Analisar uma base de operações no mercado de ações, tratando os dados, criando estatísticas e classificando o perfil dos clientes com base na quantidade de operações realizadas.

Etapas realizadas

* Carregamento do arquivo `operacoes_mercado_acoes`
* Renomeação da aba principal para `mercado_acoes`
* Ajuste dos tipos de dados das colunas
* Verificação das estatísticas das colunas
* Criação da aba `estatística`
* Cálculo de soma, mínimo, máximo, média, mediana e moda do preço
* Criação das colunas `dia`, `mês` e `ano`
* Extração dessas informações a partir da coluna `data`
* Criação da aba `cliente`
* Consulta de informações por e-mail do cliente
* Cálculo da contagem, soma e média das operações do cliente
* Classificação do perfil do cliente em:

  * Perfil Agressivo
  * Perfil Moderado
  * Perfil Conservador

Ferramentas utilizadas

* Microsoft Excel e Google Sheet
* GitHub
